# Price Drop Alert - Landing Page

Amazon Associates 심사를 위한 랜딩 페이지입니다.

## 파일 구성

- `index.html` - 메인 랜딩 페이지
- `privacy.html` - 개인정보처리방침 (Privacy Policy)
- `terms.html` - 이용약관 (Terms of Service)
- `manifest.json` - PWA 설정

## 배포 방법

### Option 1: Vercel (추천 - 가장 쉬움)

1. https://vercel.com 접속
2. GitHub 계정으로 로그인
3. "Add New" → "Project"
4. 이 폴더를 GitHub에 업로드하거나 직접 드래그&드롭
5. 자동 배포 완료!
6. 무료 도메인 제공: `your-project.vercel.app`

### Option 2: Netlify

1. https://netlify.com 접속
2. 계정 생성/로그인
3. "Add new site" → "Deploy manually"
4. 이 폴더를 드래그&드롭
5. 자동 배포!
6. 무료 도메인: `your-project.netlify.app`

### Option 3: GitHub Pages

1. GitHub에 새 리포지토리 생성
2. 이 폴더의 파일들 업로드
3. Settings → Pages → Source: main branch
4. 무료 도메인: `username.github.io/repo-name`

## Amazon Associates 등록 시

**Website URL 입력:**
```
https://your-project.vercel.app
```

또는 커스텀 도메인 구매 후:
